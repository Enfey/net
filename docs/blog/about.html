About This Site

Welcome to this digital estate.
This site intends to serve a few purposes, so it's had to take on a few forms. Below the forms this website has taken are detailed:

1. A radio
Deliverance is a radio with an intense focus on extreme music of varying form. Dark ambient, neofolk, power electronics, drone, black metal, sludge metal, experimental electronic etc are all genres you can come to expect. The first show is planned for the 18th of September, complete with live TouchDesigner visuals. If you're interested in a radio slot, please email or drop a message on Instagram.

2. Literature
Deliverance is an archive of literary creations. Note: Expect a number of these works to feature branches of mathematics such as category theory and type theory. This will require some degree of mathematical literacy, but this isn't strictly necessary to understand the primary content.

3. Sample platform
Deliverance is a repository of free-to-use samples, of varying theme.

What's coming?

This website is not complete. There'll be a few changes visually for mobile users when in the near future, but in its current form, this website isn't practical to extend further. Later on this year/early next year there'll be more significant additions to the actual feature set of the website.

Thank you for reading :)
