About This Site

Welcome to this digital estate.
This site intends to serve a few purposes, so it's had to take on a few forms[1]. Below the forms this website has taken are detailed:

1. A radio
Deliverance is a radio with an intense focus on extreme music of varying form. Dark ambient, neofolk, power electronics, drone, black metal, sludge metal, experimental electronic etc are all genres you can come to expect. The first show is planned for 5th August, complete with live TouchDesigner visuals. If you are interested in a radio slot, please email or message me on Instagram.

2. Literature
Deliverance is an archive of my personal literary creations. Note: Expect a number of these works to feature branches of mathematics such as category theory and type theory. This will require some degree of mathematical literacy, but I wouldn't say this is strictly necessary to understand the primary content.

3. Sample platform
Deliverance is a repository of free-to-use samples. I hope you'll be able to find some use for them.

What's coming?

This website is not complete. There'll be a few changes visually for mobile users when I can get round to implementing them, but in its current form, this website isn't practical to extend further. Later on this year/early next year there'll be more significant additions to the actual feature set of the website.

That's all for now. Thank you for reading :)



[1] : Just a note on thought generation function. It is not injective given that different combinations of sensory inputs and internal state can lead to the same thought. I'm sure you can think of an example for this :)
